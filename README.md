# Hangman
Classic word-guessing game implemented in Python for the terminal.

## Table of Contents
1. Overview
2. Features
3. Tech Stack   
4. Usage Instructions

## Overview  
This is a console version of Hangman where the player must guess letters of a hidden word before running out of tries. Built from scratch to demonstrate clean programming structure, game logic, user interaction and basic data handling.

## Features  
- Random word selection from a predefined list  
- Display of progress (correct letters, blanks) and number of remaining lives  
- Tracking of wrong guesses and prevention of duplicate guesses  
- Simple, modular code structure making it easy to extend  
- Runs in the terminal / command-line environment  

## Tech Stack  
- **Language:** Python 3  
- **Libraries:** Standard Library only (no external dependencies)  
- **Files included:**  
  - `hangman.py` – main game loop & logic  
  - `wordlist.txt` (or similar) – list of possible words  
  - `README.md` – project documentation
